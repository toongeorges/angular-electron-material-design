Hello, I created this project to use it as a base for all my Angular + Electron + Material Design projects and for everyone else who wants to use it.

This project is a fork of https://github.com/maximegris/angular-electron. I rely on the Angular Electron project for setting up Angular and Electron, in this fork I merely add the integration with Material Design.
